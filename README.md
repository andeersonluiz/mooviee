# Projeto: Movieee (06/2024 - 08/2024)

<p align="center">
<img src="https://github.com/andeersonluiz/mooviee/blob/main/assets/logo.png">

O site Moviee é um site criado para vizualizar filmes e series, fornecendo informaçõe como datas de lançamento, gêneros, elenco e muito mais. Desenvolvido com Next.js 14.2 e React 18.3, a aplicação utiliza Clean Architeture para modularizar o código, dividir responsabilidades e minimizar o acoplamento entre os componentes. Os dados sobre filmes e séries são obtidos através do <a href="https://developer.themoviedb.org/docs/getting-started">TMDB API</a>, consumindo via REST API. O site está disponível em inglês e português, permitindo uma experiência personalizada para os usuários de diferentes idiomas.

## Recursos
- No site, há as seguintes seções: 
  - Inicio: Apresenta os filmes e séries "Em alta hoje", com sinopses e informações adicionais. Também inclui categorias como "Em alta esta semana", "Episódios nesta semana" e "Nos cinemas", além de uma classificação dos melhores filmes e séries baseada em popularidade, avaliação e lançamentos.
  - Filmes: Exibe uma lista completa de filmes, com opções para ordenar por popularidade, título, faturamento, avaliação e data de lançamento.
  - Séries:  Lista todas séries disponíveis, permitindo ordenação por popularidade, avaliação e data de exibição.
  - Informações: Fornece detalhes sobre o filme ou série selecionado, incluindo sinopse, faixa etária, gêneros, elenco e outros dados relevantes.
  - Elenco: Mostra a lista de atores e atrizes que participaram do filme ou série, junto com a equipe técnica.
